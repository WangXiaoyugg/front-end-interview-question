1. http常见的状态码？
    - 200 请求成功
    - 301 永久重定向
    - 302 零时重定向
    - 401 未经过认证
    - 403 没有权限,禁止访问
    - 404 资源找不到
    - 500 服务器错误

2. http 请求方法
   1. GET 表示资源获取
   2. POST 表示资源创建或者提交数据
   3. PUT 表示修改资源
   4. DELETE 表示删除资源

3. http 协议中常见的Header
    - accept 代表客户端希望接受的数据类型
    - accept-encoding 浏览器发送给服务器，声明浏览器支持的编码类型
    - accept-language, 表明浏览器所支持的语言类型
    - Cache-Control, 缓存开关， no-cache表示禁用缓存
    - referer: 当前请求页面的来源页面地址，主要用于防盗链和恶意请求
